Installing

To install quacker blaster, simply run unzip the zip file to the location you want the game to reside no installation nessessary.

UnInstalling

To uninstall quacker blaster, simply run remove the folder where quacker blaster resides. Quacker Blaster is self-contained and does not modify your pc in any way.

Objective

Your goal is to shoot as many ducks as you can without missing.

Controls

Use the mouse to point the crosshair at the duck. When the crosshair and duck are lined up, press the left mouse button to fire. If you are using a lightgun, point the barrel at the duck and press the trigger.

Scoring

Each duck in rounds 1-5 are worth 100 points, in rounds 6-10 they are worth 800 points and in rounds 10 and above worth 1000 points..

Game Over

You start with 3 bullets and use 1 up everytime you fire. If you miss too many times, and run out of bullets the duck escapes. However take your time to shoot the duck and he escapes as well. If you miss too many times in a single round it's game over.

  • Round 1-10: 6 or more ducks must be hit
  • Round 11-12: 7 or more ducks must be hit
  • Round 13-14: 8 or more ducks must be hit
  • Round 15-19: 9 or more ducks must be hit
  • Round 20 and above: All 10 ducks must be hit

Troubleshooting

Error Logging

By default errors are logged to a file called error.log in the directory the game resides. If there is a problem, that file will assist you in troubleshooting the problem.

Game closes unexpectivly

One or more required resource files are missing. Make sure there is a Data file in the location the game resides and the folder contains the files background.png, bullet_icon.png, crosshair.png, duck.png, fire.wav, lunchds.ttf, silhouette.png, silhouette_hit.png and splat.wav. The error.long file montioned above will assist you in determining which file is missing. If the problem persists you can try reinstalling the application.

An error about a missing .dll is displayed on Windows

One or more of the required Dynamic Linked Library files(.dll) are missing. Make sure the files libfreetype-6.dll, libpng15-15.dll, SDL.dll, SDL_image.dll, SDL_mixer, SDL_ttf and zlib1.dll are in the same directory your game resides. The error.long file montioned above will assist you in determining which file is missing. If the problem persists you can try reinstalling the application.

No Sound

Make sure the volume isn't muted or your sound drivers are properly installed and working. If the problem presists please try reinstalling the application.

Lightgun and crosshair aren't properly aligned

The lightgun isn't calibrated, please calibrate the lightgun as per the manufacturers manual.
